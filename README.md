# 💳 Payment Gateway - Exploratory Testing Project

This project focuses on **Exploratory Testing** for a Payment Gateway system. I investigated the system's behavior and discovered defects through unscripted testing sessions.

## 🚀 What I Did:
* **Exploratory Testing:** Navigated the system to find bugs and edge cases without predefined requirements.
* **Bug Reporting:** Documented all identified defects in a professional report.
* **Test Case Design:** Created practical test cases based on my findings during exploration.
